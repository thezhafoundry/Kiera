WhatsApp Desktop setup

  1. Install VB-CABLE, then reopen this page in current Chrome or Edge on Windows.
  2. Select your physical microphone and CABLE Input as the converted output above.
  3. In WhatsApp Desktop, set Microphone to CABLE Output; keep its speaker set to your headphones.
  4. Start conversion before your call. If the relay interrupts, Keira closes it and sends no raw microphone audio.
